## Install

<Tabs syncKey="package-manager">
  <TabItem label="npm" icon="seti:npm">
    ```sh frame=none
    npm install --save typegpu typed-binary
    ```
  </TabItem>
  <TabItem label="pnpm" icon="pnpm">
    ```sh frame=none
    pnpm add typegpu typed-binary
    ```
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
    ```sh frame=none
    yarn add typegpu typed-binary
    ```
  </TabItem>
